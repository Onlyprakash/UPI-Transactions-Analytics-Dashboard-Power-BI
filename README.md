# 📌 Project Overview

This project provides an interactive transaction analytics experience built using Power BI.

The dashboard allows users to:

✔ Explore monthly transaction patterns  
✔ Monitor balance fluctuations  
✔ Analyze payment activity  
✔ Filter transactions dynamically  
✔ Compare multiple business dimensions

# 🎯 Business Problem

Digital payments generate large transaction datasets.

This dashboard helps answer:
- Which months have higher activity?
- How balances fluctuate over time?
- Which dimensions affect transactions?
- How payment behavior changes?
